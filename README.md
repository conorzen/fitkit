# WatchFit

A watchOS fitness app that leverages WorkoutKit to deliver seamless workout experiences on Apple Watch.

## Overview

WatchFit integrates with Apple's WorkoutKit framework to provide accurate workout tracking, custom workout types, and real-time metrics on your Apple Watch.

## Features

- Custom workout types and routines
- Real-time heart rate monitoring
- Workout metrics tracking (distance, pace, calories)
- Integration with Apple Health
- Live workout statistics
- Post-workout summaries and analytics

## Requirements

- watchOS 10.0+
- Xcode 15.0+
- iOS 17.0+ (for companion app)
